# portfolio
## 📖 About the project
This is a project that I made while learning React, to create my first portfolio website. It shows who I am, my projects, skills and social-medias. It's responsive for mobile and also has a toggle light/dark mode. It was supposed to create a two languages website, with a toggle, but due GitHub Pages problems, it's on the code but not on the website hosted.

## 🧔 User Story
It's a portfolio page, where the user can see infos about me, my "about me" section, my previous projects, my skills and contact section, where the user can access my social-medias or send me an e-mail. Also, the website is responsive, with a layout just for mobile users.
